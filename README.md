# 🏦 AI Loan Approval System

An end-to-end Machine Learning project that predicts loan approval using applicant financial and demographic data.

## 📌 Project Overview

This project demonstrates a complete ML pipeline:

- Data preprocessing
- Feature engineering
- Model training & evaluation
- Hyperparameter tuning
- Model serialization
- Streamlit-based web deployment

The system predicts whether a loan should be approved or rejected based on risk factors.

---

## 🧠 Machine Learning Concepts Used

- Supervised Learning (Binary Classification)
- Logistic Regression
- Decision Tree
- Random Forest
- Feature Scaling
- Confusion Matrix
- ROC-AUC Score
- Model Serialization (joblib)

---

## 🛠 Tech Stack

- Python
- Scikit-Learn
- Pandas
- Streamlit
- Matplotlib

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Business Application

This project simulates a real-world FinTech loan risk system used in:
